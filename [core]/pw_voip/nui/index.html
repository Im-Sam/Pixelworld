TokoVOIP Required
PixelWorld uses TokoVOIP as a in-game replacement to voice communications, the primary reason for this is for the compatabilities with directional audio, which helps with the realism that you are talking to someone,
Aswell as directional audio, there are some other enhancements that are made avaliable including the following
Better voice quality
Radio system
Proximity chat unique to each player
Phone calls
Radio clicks
Radio effects
